筆記

1.至 Leo-editor GitHub
2.點選 releases 下載zip檔
3.將載下來的檔案拉到 Ana3_2015 的 data
4.下載老師給的fix資料夾並拉到Ana3_2015
6.開啟SciTE
7.將fix資料夾內的fix.bat及start.bat拉入SciTE
8.fix.bat需新增 copy leo-editor-broke-abbrev.zip E:\Ana3_2015\data\ (我的檔案是放在E槽的隨身碟)
9.start.bat把第25行改為 pip install leo-editor-broke-abbrev.zip
10.儲存完之後點兩下fix.bat
11.再啟動 start 就安裝成功了
